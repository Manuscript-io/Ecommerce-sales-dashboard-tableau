# Ecommerce Sales Dashboard | Tableau
### Project Overview
This project presents an interactive Ecommerce Sales Dashboard built using Tableau to analyze Year-to-Date (YTD) sales performance across different markets, customer segments, and product categories.

The dashboard helps businesses track key performance indicators, identify growth opportunities, and monitor customer acquisition trends.

- Camping & Hiking and Water Sports categories show positive growth.
- Several product categories have experienced a decline compared to the previous year.
- Customer acquisition trends vary significantly across markets.
